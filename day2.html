Working with HTML Elements and Styling with CSS

HTML: Lists, Tables, and Images

Today, we will learn how to create lists, tables, and insert images in HTML.

Lists

There are two main types of lists in HTML:

  • Ordered List: Numbered items.
  • Unordered List: Bulleted items.
<h3>Ordered List</h3>
<ol>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>

<h3>Unordered List</h3>
<ul>
  <li>Item A</li>
  <li>Item B</li>
  <li>Item C</li>
</ul>

Tables

Tables are used to display data in rows and columns.

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td>30</td>
  </tr>
</table>

Images

Images can be added using the <img> tag.

<img src="https://via.placeholder.com/150" alt="Placeholder Image">

HTML: Introducing the <a> Tag and href Attribute

The <a> tag is used to create hyperlinks, which allow users to navigate to other pages or resources. The href attribute specifies the URL of the page the link goes to.

Example: Linking to an External Website

<a href="https://www.example.com" target="_blank">Visit Example Website</a>

Example: Linking to an Internal Page

<a href="about.html">About Us</a>

You can also use href="#section-id" to link to a specific section of the same page.

CSS: Box Model and Styling Elements

CSS Box Model includes margins, borders, padding, and the content area. Here's how it looks:

Here are some common CSS properties to style elements:

Property Description Example
color Sets the text color. color: red;
background-color Sets the background color. background-color: yellow;
margin Creates space around an element. margin: 20px;
padding Creates space inside an element. padding: 10px;

Simple CSS Example

Here's a simple CSS snippet to style a webpage:

body {
  background-color: lightgray;
  color: black;
  font-family: Verdana, sans-serif;
}
h1 {
  color: darkblue;
  text-align: center;
}
p {
  padding: 10px;
  border: 1px solid #ccc;
}
