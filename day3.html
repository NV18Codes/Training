HTML: Forms and Input Types

Forms allow users to input and submit data. Here are some examples:

Basic Form

<form action="/submit" method="POST">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>

  <button type="submit">Submit</button>
</form>

Input Types

Here are some common input types:

  • Text: <input type="text">
  • Email: <input type="email">
  • Password: <input type="password">
  • Checkbox: <input type="checkbox">
  • Radio: <input type="radio">
  • Button: <input type="button">

Dropdown and Textarea

<label for="country">Country:</label>
<select id="country" name="country">
  <option value="india">India</option>
  <option value="usa">USA</option>
  <option value="uk">UK</option>
</select>

<label for="message">Message:</label>
<textarea id="message" name="message" rows="4" cols="50"></textarea>

HTML: Semantic Elements

Semantic elements provide better structure and accessibility. Examples include:

  • <header>: Defines a page or section header.
  • <footer>: Defines a page or section footer.
  • <main>: Main content area.
  • <article>: Self-contained content like a blog post.
  • <section>: Thematic grouping of content.
  • <aside>: Sidebar or related content.
  • <nav>: Navigation links.

Example of semantic structure:

<header>
  <h1>Welcome to My Website</h1>
</header>

<main>
  <article>
    <h2>Article Title</h2>
    <p>This is an article.</p>
  </article>
</main>

<footer>

</footer>

CSS: Advanced Positioning

CSS positioning allows precise placement of elements:

  • Static: Default positioning.
  • Relative: Positioned relative to its normal position.
  • Absolute: Positioned relative to its nearest positioned ancestor.
  • Fixed: Positioned relative to the viewport.

Example CSS:

.box {
  width: 100px;
  height: 100px;
  background: coral;
}

.relative {
  position: relative;
  top: 20px;
  left: 20px;
}

.absolute {
  position: absolute;
  top: 50px;
  left: 50px;
}

.fixed {
  position: fixed;
  top: 10px;
  right: 10px;
}

HTML: Audio and Video

HTML allows embedding multimedia content like audio and video. Examples:

Audio Element

<audio controls>
  <source src="audio-file.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

Video Element

<video controls width="320" height="240">
  <source src="video-file.mp4" type="video/mp4">
  Your browser does not support the video element.
</video>

HTML: Using <div> and <section>

The <div> element is a container for grouping content, while <section> is used for grouping semantically related content.

Example Layout

<div class="container">
  <header>Header Content</header>
  <div class="main-content">
    <section>Section 1 Content</section>
    <section>Section 2 Content</section>
  </div>
  <footer>Footer Content</footer>
</div>

Homework

Apply what you've learned from Days 1, 2, and 3 by creating a fully structured webpage:

  • HTML:
    • Use semantic elements like <header>, <main>, <section>, and <footer>.
    • Create a form with various input types: text, email, password, number, checkbox, and radio.
    • Include a dropdown menu and a textarea for user input.
    • Add a navigation bar with links that point to different sections of the page.
    • Use the <audio> and <video> elements to embed multimedia content.
  • CSS:
    • Style the page with a consistent color scheme and font family.
    • Use CSS to create a layout with a header, sidebar, and main content area.
    • Experiment with different positioning techniques: static, relative, absolute, and fixed.
    • Add a marquee to display important announcements or updates.
  • Bonus Task:
    • Design a simple card layout for displaying information (e.g., user profiles, products, or articles).
    • Include hover effects on buttons and navigation links for better interactivity.
