Day 1: Getting Started with HTML and CSS

What is Frontend Development?

Frontend development involves creating the user interface and experience of a website using technologies like HTML, CSS, and JavaScript.

HTML Basics

HTML (HyperText Markup Language) is used to structure the content of a webpage. Here is a simple example:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Welcome!</h1>
    <p>This is my first webpage.</p>
  </body>
</html>

CSS Basics

CSS (Cascading Style Sheets) is used to style the HTML content. Here's how you can add some style:

<style>
  body {
    background-color: lightblue;
    font-family: Arial, sans-serif;
  }
  h1 {
    color: darkblue;
  }
</style>
